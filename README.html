Introduction

rvalidate computes validation statistics for models and draw up aesthetic tables and plots. Supports error, coverage, and Probability integral transforms. Input data should be formated as our simulated data. See rvalidate::rv_data below

library(rvalidate)

Statistics for data with multiple observations within a grouping

For data containing multiple observations per group (e.g.Â country-year you) we compute statistics for all unique permutations of the data where there is only one observation from each group in each permutation. Note the run time will increase substantially as the number of multiple observations increase.
